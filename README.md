# mtg-setup
Автоматизация установки mtproto для телеграмм 

Данная ветка представляет собой гайд по развертыванию mtproto на своем сервере, для платформы telegram за основу взят mtproto c FakeTLS.

Расположен здесь: ghcr.io/9seconds/mtg:stable

Документация к нему: https://pkg.go.dev/github.com/9seconds/mtg/v2/mtglib#section-documentation

Изначальный гайд размещен на платформе Reddit и данный репозиторий является ничем иным как сохранение фаилов автоматизации.
